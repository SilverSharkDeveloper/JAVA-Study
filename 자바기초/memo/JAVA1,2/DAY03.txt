형변환(type casting)
		-자동 형변환
				정수 + 정수 = 정수
				정수 + 실수 = 실수
		-강제 형변환
				(자료형) 값
				(int)3.14 -> 3


1.다른 자료형을 문자열
		문자열과 다른 자료형을 연결하면 결과는 항상 문자열이 된다.  빈문자열 이용 (여러가지 방법이 존재)

2. 문자열을 다른 자료형으로
	기본 자료형은 기본 자료형끼리 형변환이 된다.
	클래스 타입은 기본 자료형으로 형변환이 불가능하다.
	그러나 String 타입은 Integer, Double, Boolean 클래스의 도움을 받아서 형변환을 할 수 있다.

함수
	어떤 기능을 하는 것
	이름 뒤에 ()가 있다.
메소드
	함수와 같다. (클래스 안에서의 함수)
	JAVA에서 모든 함수는 클래스 내부에 만들어지므로 메소드라고 말한다.

입력 클래스 
	Scanner sc = new Scanner(System.in);
	자료형 변수명      값


입력 메소드
	Scanner 클래스 내부에 입력 메소드가 만들어져있다.

	next() : 입력 받은 값을 String 타입으로 돌려준다.
		입력한 값을 띄어쓰기, 엔터로 구분하고 분리한다. -----------엔터또는 띄어쓰기전까지만 가져온다.
	nextLine() : 입력 받은 값을 String 타입으로 돌려준다.
		띄어쓰기를 포함한 한 줄 전체를 입력받는다.

	nextint() : 입력 받은 값을 int타입으로 돌려준다.
		입력한 값을 띄어쓰기, 엔터로 구분하고 분리한다.
	nextdouble() : 입력 받은 값을 double타입으로 돌려준다
		입력한 값을 띄어쓰기, 엔터로 구분하고 분리한다.


	next(), nextint(), nextdouble()을 사용하면 엔터 값이 남아있게 되어 다음에 nextLine을 쓸경우 남아있는 엔터가 들어가 입력
	받지 못하므로 입력받기전에 nextLine()을 사용해 엔터를 소모시켜야 한다.

연산자
	기능이 있는 특수 문자
연산자의 우선순위
	하나의 식에 여러 개의 연산자를 사용하면 어떤 연산이 먼저 되는지를 구분한다.
	
	최	-	최우선 연산자          ()
	단	-	단항 연산자	
	산	-	산술 연산자
	쉬	-	쉬프트 연산자
	관	-	관계 연산자
	리	-	논리 연산자
	삼	-	삼항 연산자
	대	-	대입 연산자


결합성
	하나의 수식에 동일한 연산자가 여러 개 사용되몀 알맞은 방향으로 결합되어 연산되는 성질

산술 연산자
	+	덧셈
	-	뺄셈
	*	곱셈
	/	나눗셈
	%	나머지 연산자
