프로젝트
	하나의 프로그램의 단위 (일반적인) 
	프로그램을 구성하는 것들을 모아둔 폴더

패키지
	소스 파일들의 묶음 단위
	프로젝트 하위 폴더

 
소스 파일
	.java로 끝나는 파일이며 소스코드가 작성되어 있다.	(.뒤에오는 건 확장자를 알려준다.)
	이클림스에서 클래스를 생성하면 자동으로 클래스와 같은 이름으로 소스파일을 만들어준다.

클래스
	자바의 모든 코드는 클래스 내부에 작성된다.
	크래스의 이름은 반드시 대문자로 시작해야한다. (만들어지긴하는데 개발자들끼리에 암묵적인 룰이다.) - 면접가면 떨어짐 ㅋ;

중괄호{}
	자바에서는 중괄호{}로 영역을 표시한다.

메소드
	이름뒤에 소괄호()가 있는 대부분의 명령어는 메소드 이다.
	메소드는 어떠한 기능이다.

메인 메소드
	자바 프로그램이 실행되면 메인 메소드의 코드가 실행된다.
	프로그램이 시작되는 시작점 (entry point)
	
키워드 (예약어)
	이미 정해진 역할이 있는 단어

System.out.print()  <syso ctrl + space>

System 대문자로 시작 - 클래스
. = ~안에
print() - 메소드 = 함수

출력 메소드
	print()
	소괄호 안에 있는 것(값)을 콘솔창에 출력한다.

닷 연산자, 하위 연산자
	.(점)
	~안에 라고 해석하면 된다.

세미콜론 ; 
	자바에서는 세미콜론; 으로 문장의 끝을 표시한다.

들여쓰기
	자바에서 들여쓰기는 코드에 영향을 주지 않는다. 
	가독성을 높이기 위해 사용한다

	=중괄호와 메소드를 같은 세로줄로 맞춰준다.

변수(variable)란?
	어떤 값을 저장하기 위한 저장 공간

변수의 특징
	1.변수는 오직 하나의 자료형(date type)만 저장할 수 있다.
	2.변수는 오직 하나의 값만 저장할 수 있다.

정리
	변수란 하나의 자료형과 하나의 값만 저장할 수 있는 저장공간이다.

변수의 선언과 사용
	-변수의 선언(변수를 만든다)
	자료형 변수명;
	int a;
	
	-변수에 값을 저장하는 방법
	변수명 = 값
	대입연산자(=) : 오른쪽의 값을 왼쪽에 저장한다.
	a = 3;

	-선언과 동시에 초기화(처음으로 값을 저장하는 것)
	자료형 변수명 = 값;
	int a = 3;
	
	-변수에 저장된 값을 사용하는 방법
	변수명을 이용한다.
	
	변수가 저장공간으로 사용되는 경우
	-변수를 선언할때
	-대입연산자 왼쪽에 오는 변수
	나머지경우는 변수는 값으로 해석된다.

	-여러개의 변수를 선언하는 방법
	자료형 변수명1, 변수명2, 변수명3

*정리
	- 변수의 선언
	int a;
	int a, b, c;
	
	-변수의 사용
	저장공간으로 사용
		int a;
		a=10;
	값으로 사용
		위 두가지 경우 말고 대부분 값으로 사용

변수 선언시 주의사항
	같은 영역에서 같은 이름의 변수를 선언할 수 없다.
	되도록 소스코드 위쪽에 한꺼번에 선언한다.
	되도록 선언과 동시에 초기화를 한다.

변수명 주의사항
	-문자부터 시작한다.
	-특수문자를 쓸 수 없다.(단 , _ $는 가능)
	-공백을 사용할 수 없다  ----쓰고싶은경우_를 사용한다.	
	-키워드(예약어)를 사용할 수 없다
	-되도록 소문자로 시작하자(대소문자를 구분한다.) ---대문자를 사용할경우 코드읽는자가 클래스로 착각할 수 있다.
	-되도록 의미 없는 문자를 사용하지 않는다.-------가독성이 떨어진다.
	-되도록 한글은 사용하지 않는다.
	
	userage : 알아보기 힘들다
	user_age : 스네이크 표기법, 팟홀 표기법
	userAge : 카멜표기법 (중간에 단어가 바뀔때 대문자로 바꿈)

변수를 사용하는 이유 (값을 왜 변수에 저장해서 사용하는가?)
	1. 반복되는 값을 쉽게 관리하기 위해서(값을 재사용하기 편함)
	2. 값의 의미 부여를 하기 위해서

RAM(random access memorry)
	int a;
	a라는 이름의 저장공간이 RAM에 할당된다.
주소값
	메모리(RAM)에 저장공간이 할당되면 고유한 값인 주소값이 부여된다.

제어문자(Escape Character)
		-따옴표 안에서 사용되며 미리 예약된 특수한 기능을 하는 문자
		-\ (역슬래쉬)를 사용한다.
		
		-\n : 줄바꿈
		-\t : 일정 간격을 띄워준다.(tab)
		-\" : 문자열 안에 큰 따옴표를 표현
		-\' : 문자열 안에 작은따옴표를 표현
		-\\ : 문자열 안에 역슬래시를 표현한다.

출력메소드의 종류
	괄호 안에 있는 값을 콘솔창에 출력함
	println()
		값을 넣으면 출력하고 줄바꿈이 된다.
	print()
		값을 넣으면 출력하고 줄바꿈이 안된다.
	printf()
		형식(서식 format)에 맞는 값을 출력하고 줄바꿈이 안된다
형식 지정자(서식 문자)
		-출력 형식을 지정한다.
		-%로 시작하며 printf()에서 사용한다.
		-반드시 따옴표 안에서 작성한다.
		-','쉼표 뒤에 나오는 값이 형식 지정자 위치로 들어가게된다.
		

		자료형(type)		형식지정자(서식문자)
		int			%d ->decimal(10진수 정수)
		double		%f  ->float(실수)
		char			%c ->character(문자)
		String			%s ->string(문자열)
